WHO IS ANDREW BALDOCK?

About Me

Hi! I'm a design-savvy web developer seeking a JavaScript focused position.

I started building web pages in 1995, eventually launching a new website for UC Berkeley in 2000. This website stood for 8 years: an eternity in internet time. During that time I went to work for Macys.com to help build their retail website. Most recently I worked on the Mindjet.com website where I built out the signup flows and user try/buy funnels.

I love JavaScript: building beautiful, visual, world-class web applications is what I want to be doing.

"One of the most passionate and committed developers I've ever worked with... a joy to work with and his team adores him!"

      —Jascha Kaykas-Wolff CMO, Mindjet

Skills

JavaScript, HTML5, CSS3, jQuery, able to work in any JavaScript project or framework; able to code vanilla JS without jQuery. Photoshop wiz. Apache configuration. Some PHP, interested in more. Some JSP. Skilled at page optimization and load times.
